# JUNYA ISHIHARA PERSONAL TRAINING OFFICIAL SITE

このプロジェクトは、プロフェッショナルパーソナルトレーナー **石原淳哉 (J. ISHIHARA)** のオフィシャルサイト兼ブランディングメディアです。

## 🌟 Brand Identity: "Deep Navy & Premium Minimalist"

本サイトは、これまでの派手なオレンジやゴールドのデザインを完全に卒業し、**「Deep Navy / Zinc / Pure White」**を中心とした、極めて洗練された高級パーソナルブランディングを展開しています。

### Core Design System
- **Base Color**: `#FFFFFF` (Solid White)
- **Primary Color**: `#262636` (Deep Navy / `navy-500`)
- **Accent Color**: `#52525B` (Zinc-600 / `zinc-600`)
- **Typography**: `Outfit` (Headings) / `Inter` (Body Content)

### Visual Philosophy
- **Minimalism**: 余分な装飾を削ぎ落とし、余白（Negative Space）を贅沢に使うことで、石原様のプロフェッショナリズムと信頼性を際立たせます。
- **Monotone Fidelity**: 基本はモノトーンとネイビー。彩度を抑えることで、高品質な写真素材の力強さを最大化します。
- **Sophisticated UI**: Remix Iconによる知的で繊細なアイコンセットと、マイクロインタラクションによるプレミアムなユーザー体験。

## 🚀 Teck Stack
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **CMS**: Sanity.io (Headless CMS)
- **AI Engine**: Advanced Content Asset Analytics (Custom SEO & Tag Engine)
- **Infrastructure**: Vercel (Deployment)

## 📁 Key Components
- **Blog Architecture**: 高度なポータブルテキストエンジンによる記事レンダリング。
- **AI Asset Engine**: 記事内容を分析し、最適なメタデータとタグを自動生成するインテリジェントな管理画面。
- **Premium Pages**: 各サービス（In-Studio / Visiting）、プロフィール、料金プランに至るまで、完全に統一されたブランドエクスペリエンスを提供。
