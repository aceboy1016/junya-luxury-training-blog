# JUNYA ISHIHARA PERSONAL TRAINING OFFICIAL SITE

このプロジェクトは、プロフェッショナルパーソナルトレーナー **石原淳哉 (J. ISHIHARA)** のオフィシャルサイト兼ブランディングメディアです。

---

## 🌟 Brand Identity: "Deep Navy & Premium Minimalist"

本サイトは、GitHubの履歴（`0c4779c` / `a4abaca`）に刻まれた **「全10セクションの膨大なコンテンツ」** を完全に継承しつつ、**「Deep Navy / Zinc / Pure White」** を中心とした最高峰の高級パーソナルブランディングへと昇華させています。

### Core Design System
- **Base Color**: `#FFFFFF` (Solid White)
- **Primary Color**: `#262636` (Deep Navy / `navy-500`)
- **Typography Icons**: `Outfit` (Headings) / `Inter` (Body) / `Remix Icon 3.5`

### Component Restoration Assets
1. **Hero**: 魂のキャッチコピー「変化を実感するから自然と続けたくなる」の復刻。
2. **Methodology**: 詳細な4ステップ・アセスメント・メソッド。
3. **Why Me**: 3つの「選ばれる理由」と段階的アプローチ。
4. **Concept**: 科学的メソッド × 最高級サービスの融合図解。
5. **Service Flow**: 迷わせないための初回案内フロー。
6. **Locations**: HALLEL半蔵門・恵比寿の両拠点情報。
7. **Pricing**: 細分化された通常・出張セッションの料金体系。
8. **Testimonials**: 実際に変化を感じたお客様の声と信頼の統計。
9. **Profile**: 「ホネカワスネオ」から始まるストーリー、詳細経歴、スキルバー。
10. **Contact**: Google Form / LINE / Mail のトリプルアンカー。

---

## 🚀 Technical Architecture
- **Framework**: Next.js 16 (App Router)
- **CMS**: Sanity.io (ブログ記事・資産管理)
- **Restoration Policy**: GitHubの過去コミットを正味の「正解」とし、デザインのみを最新化するマージ戦略。

---

## 🔧 開発ガイド

```bash
# 依存関係インストール
npm install --legacy-peer-deps

# 開発サーバー
npm run dev

# GitHubへの同期
git checkout main
git merge restore/original-content-navy-design
git push origin main
```
